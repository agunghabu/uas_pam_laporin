# [EN] Laporin
Campus facility damage reporting application.

## About
Laporin is an application for reporting facility damage within the campus environment.
With 2 types of users: User (reporter) and Admin (report manager).

### Main Features
- Capture damage photos directly from the camera
- Submit reports with title and description
- Select location (Area & Unit)
- Track report status
- Export reports to CSV (Admin)
- App theme customization

## How to Use
### User
1. Login using registered ID and password
2. "Submit Report" tab:
   - Tap the photo area to capture the damage image
   - Fill in the report title
   - Add description (optional)
   - Select Area and Unit of the damage location
   - Tap the Submit Report button
3. "My Reports" tab:
   - View the list of submitted reports
   - Track report status (Pending/Active/Completed/Rejected)
4. Tap the menu icon to access theme settings and logout

### Admin
1. Login using admin account
2. View reports by status:
   - Pending: New reports that haven't been processed
   - Active: Reports currently being handled
   - Completed: Reports that have been resolved
3. Manage reports:
   - On Pending tab: Accept or reject reports
   - On Active tab: Reject reports or mark as completed
4. Tap the save icon to export all reports to CSV file
5. Tap the menu icon to access theme settings and logout

# [ID] Laporin
Aplikasi pelaporan kerusakan fasilitas kampus.

## Tentang Aplikasi
Laporin adalah aplikasi untuk melaporkan kerusakan fasilitas di lingkungan kampus.
Dengan 2 jenis pengguna: User (pelapor) dan Admin (pengelola laporan).

### Fitur Utama
- Ambil foto kerusakan langsung dari kamera
- Submit laporan dengan judul dan deskripsi
- Pilih lokasi (Area & Unit)
- Pantau status laporan
- Export laporan ke CSV (Admin)
- Tema aplikasi

## Cara Penggunaan
### User
1. Login menggunakan ID dan password yang terdaftar
2. Tab "Submit Report":
   - Tap area foto untuk mengambil gambar kerusakan
   - Isi judul laporan
   - Tambahkan deskripsi (opsional)
   - Pilih Area dan Unit lokasi kerusakan
   - Tap tombol Submit Report
3. Tab "My Reports":
   - Lihat daftar laporan yang sudah disubmit
   - Pantau status laporan (Pending/Active/Completed/Rejected)
4. Tap ikon menu untuk mengakses pengaturan tema dan logout

### Admin
1. Login menggunakan akun admin
2. Lihat laporan berdasarkan status:
   - Pending: Laporan baru yang belum diproses
   - Active: Laporan yang sedang ditangani
   - Completed: Laporan yang sudah selesai
3. Kelola laporan:
   - Pada tab Pending: Terima atau tolak laporan
   - Pada tab Active: Tolak laporan atau Tandai sebagai selesai
4. Tap ikon save untuk export semua laporan ke file CSV
5. Tap ikon menu untuk mengakses pengaturan tema dan logout
